# Drosse
![Drosse](./Drosse.svg "Dynamic js mock server")

## Features
- Cascading Proxies
- Fallback to static JSON mocks
- Skip login/session
- Seamless integration to your project OR
- Standalone global mode

## Future features
- In-memory db
- Persist state (in-memory db dump to JSON file)
- Hateoas support
- CLI + UI
- Journey
- Sync with OpenAPI (Swagger)
- GraphQL support ?
